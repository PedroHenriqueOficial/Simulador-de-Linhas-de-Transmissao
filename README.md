# Simulador-de-Linhas-de-Transmissao
O presente repositório tem como objetivo apresentar o desenvolvimento de um simulador de linhas de transmissão. O simulador desenvolvido busca simular o funcionamento das linhas de transmissão em corrente alternada, sinais, por meio de ondas estacionárias, gráficos e desenhos esquemáticos, no domínio da frequência bem como a carta de Smith.
